SatelliteDataSources.AbstractSatellite — Type

The supertype of all sensor types. Provides sensor-specific information to many RemoteSensingToolbox methods.

source
SatelliteDataSources.Landsat8 — Type

Implements the AbstractSatellite interface for Landsat 8.

Supported layers are: (:B1, :B2, :B3, :B4, :B5, :B6, :B7, :blue, :green, :red, :nir, :swir1, :swir2, :QA, :bands).
